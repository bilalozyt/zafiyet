Kullanıcı Arama Uygulaması (Güvenlik Açığı Sürümü)

Uyarı: Bu uygulama güvenlik açıkları içermektedir!

Bu uygulama SQL Enjeksiyon güvenlik açığını göstermektedir. Sadece eğitim amaçlıdır.

Bu uygulama, veritabanında kullanıcı aramanıza olanak tanır. Arama, arama terimiyle eşleşen kullanıcı adlarını arar.




SQL Enjeksiyon Örnekleri:

  • ' OR '1'='1 - Tüm kullanıcıları getir
  • ' OR 1=1 -- - Tüm kullanıcıları getir (alternatif sözdizimi)
  • ' UNION SELECT 1, sql, 3, 4 FROM sqlite_master -- - Veritabanı şemasını göster
